## 一、指南概述

中国营养学会于2022年6月1日发布《中国婴幼儿喂养指南（2022）》，是《中国居民膳食指南（2022）》的重要组成部分。该指南根据不同年龄段儿童的生理特点和营养需求，分为三个独立部分：

| 指南名称                    | 适用年龄     | 核心准则/推荐数 |
| :-------------------------- | :----------- | :-------------- |
| 《0-6月龄婴儿母乳喂养指南》 | 出生-180天   | 6条准则         |
| 《7-24月龄婴幼儿喂养指南》  | 6月龄-24月龄 | 6条准则         |
| 《学龄前儿童膳食指南》      | 2-6岁        | 5条核心推荐     |

------

## 二、0-6月龄婴儿母乳喂养指南

### 核心原则

6月龄内是人一生中生长发育的第一个高峰期，对能量和营养素的需要相对高于其他任何时期。母乳既可提供优质、全面、充足和结构适宜的营养素，又能完美适应婴儿尚未成熟的消化能力。

### 准则1：母乳是婴儿最理想的食物，坚持6月龄内纯母乳喂养

- 纯母乳喂养能满足6月龄内婴儿所需要的全部能量、营养素和水
- 婴儿出生后不要喂任何母乳以外的食物
- 坚持让婴儿直接吸吮母乳，只要母婴不分开，就不用奶瓶喂哺人工挤出的母乳

### 准则2：生后1小时内开奶，重视尽早吸吮

- 分娩后母婴即刻开始不间断的肌肤接触
- 新生儿出生后30分钟到1小时内的吸吮有助于建立早期母乳喂养
- 生后体重下降只要不超过出生体重的7%就应坚持纯母乳喂养
- 婴儿吸吮前不需过分擦拭或消毒乳房

### 准则3：回应式喂养，建立良好的生活规律

- 及时识别婴儿饥饿及饱腹信号，哭闹是饥饿信号的最晚表现
- 按需喂养，不要强求喂奶次数和时间，但生后最初阶段会在10次以上
- 婴儿异常哭闹时，应考虑非饥饿原因
- 一般2月龄后可逐渐规律哺乳

### 准则4：适当补充维生素D，母乳喂养无需补钙

- 纯母乳喂养的婴儿出生后数日开始每日补充维生素D 10μg（400 IU）
- 纯母乳喂养的婴儿不需要补钙
- 出生后应注意补充维生素K

### 准则5：任何动摇母乳喂养的想法和举动都必须咨询医生或其他专业人员

- 绝大多数母亲都能纯母乳喂养自己的孩子
- 母乳喂养遇到困难时，需要医生和专业人员的支持
- 任何婴儿配方奶都不能与母乳相媲美，只能作为母乳喂养失败后的无奈选择
- 不要直接用普通液态奶、成人和普通儿童奶粉、蛋白粉、豆奶粉等喂养6月龄内婴儿

### 准则6：定期监测婴儿体格指标，保持健康生长

- 6月龄内婴儿每月测量一次身长、体重和头围
- 选用国家卫生标准《5岁以下儿童生长状况判定》（WS/T423-2013）判断生长状况
- 出生体重正常婴儿的最佳生长模式是基本维持其出生时在群体中的分布水平

### 喂养要点速查表

| 项目         | 建议                      |
| :----------- | :------------------------ |
| 喂养方式     | 纯母乳喂养                |
| 喂养频率     | 按需喂养，24小时8-10次    |
| 维生素D补充  | 出生后数日开始，400 IU/天 |
| 钙剂补充     | 不需要                    |
| 体格监测     | 每月1次                   |
| 首次开奶时间 | 生后1小时内               |

## 三、7-24月龄婴幼儿喂养指南

### 核心原则

满6月龄后，单一的母乳喂养已经不能完全满足婴儿对能量及营养素的需求，必须引入其他营养丰富的食物。这一阶段是从被动接受喂养转变到自主进食的关键期。

### 准则1：继续母乳喂养，满6月龄起必须添加辅食，从富含铁的泥糊状食物开始

- 婴儿满6月龄后继续母乳喂养到2岁或以上
- 辅食添加过早或过晚都会影响健康
- 首先添加肉泥、肝泥、强化铁的婴儿谷粉等富铁的泥糊状食物
- 有特殊需要时须在医生的指导下调整辅食添加时间

### 准则2：及时引入多样化食物，重视动物性食物的添加

- 每次只引入一种新的食物，逐步达到食物多样化
- 不盲目回避易过敏食物（如鸡蛋、小麦、鱼、坚果等），1岁内适时引入各种食物
- 从泥糊状食物开始，逐渐过渡到颗粒状、半固体、固体食物
- 逐渐增加辅食频次和进食量

### 准则3：尽量少加糖盐，油脂适当，保持食物原味

- 婴幼儿辅食应单独制作
- 保持食物原味，尽量少加糖、盐及各种调味品
- 辅食应含有适量油脂——婴幼儿需要适量的油脂提供生长所需能量
- 1岁以后逐渐尝试淡口味的家庭膳食

### 准则4：提倡回应式喂养，鼓励但不强迫进食

- 进餐时父母或喂养者与婴幼儿应有充分的交流，识别其饥饱信号
- 饥饿信号：张嘴、扑向食物；饱腹信号：扭头、闭嘴
- 耐心喂养，鼓励进食，但绝不强迫喂养
- 鼓励并协助婴幼儿自主进食，培养进餐兴趣
- 进餐时不看电视，不玩玩具，每次进餐时间不超过20分钟

### 准则5：注重饮食卫生和进食安全

- 选择安全、优质、新鲜的食材
- 制作过程始终保持清洁卫生，生熟分开
- 不吃剩饭，妥善保存和处理剩余食物
- 饭前洗手，进食时应有成人看护
- 注意剔除骨头和鱼刺，不吃整粒花生、坚果、果冻等以防发生进食意外

### 准则6：定期监测体格指标，追求健康生长

- 每3个月测量一次身长、体重、头围等体格生长指标
- 平稳生长是婴幼儿最佳的生长模式
- 鼓励婴幼儿爬行、自由活动，增加身体活动

### 辅食添加阶段要点

| 月龄      | 食物质地     | 新增要点           |
| :-------- | :----------- | :----------------- |
| 6月龄     | 泥糊状       | 强化铁食物优先     |
| 7-9月龄   | 颗粒状/末状  | 逐步增加食物种类   |
| 10-12月龄 | 块状/指状    | 鼓励自主进食       |
| 12-24月龄 | 接近家庭饮食 | 尝试淡口味家庭膳食 |

## 四、学龄前儿童膳食指南（2-5岁）

### 核心原则

2-5岁学龄前儿童的生长发育和营养状况直接关系青少年期及成人期肥胖及慢性病风险。此阶段食物种类和膳食模式已接近成人，但消化能力仍有差异。

### 在《中国居民膳食指南(2022)》平衡膳食八条准则基础上，增加以下5条核心推荐：

### 核心推荐1：食物多样，规律就餐，自主进食，培养健康饮食行为

- 每日膳食应安排3次正餐+2次加餐，加餐在上、下午各1次
- 鼓励儿童自主进食，培养进餐兴趣
- 父母应保持自身良好的进餐习惯，成为婴幼儿的榜样

### 核心推荐2：每天饮奶，足量饮水，合理选择零食

- 我国儿童钙摄入量普遍偏低，应鼓励天天饮奶
- 建议每天饮奶量350-500ml或相当量奶制品
- 建议每日饮水量：2-3岁600-700ml，4-5岁700-800ml
- 少量多次饮用（上、下午各2-3次）
- 合理选择零食，不喝含糖饮料

### 核心推荐3：合理烹调，少调料，少油炸

- 从小培养清淡口味
- 减少高盐、高糖、高脂食物的摄入
- 多采用蒸、煮、炖等烹饪方式

### 核心推荐4：参与食物选择与制作，增进对食物的认知和喜爱

- 鼓励儿童参与家庭食物选择和制作过程
- 吸引儿童对各种食物的兴趣
- 享受烹饪乐趣，爱惜食物

### 核心推荐5：经常户外活动，定期体格测量，保障健康成长

**身体活动建议**：

- 每天身体活动总时间：>180分钟
- 每天户外活动时间：≥120分钟
- 每天中高强度身体活动：≥60分钟

**视屏时间限制**：

- 每天视屏时间累计：<1小时，且越少越好

**体格监测**：

- 定期监测儿童身高体重，每半年1次

### 学龄前儿童每日膳食建议

| 食物类别        | 推荐量          |
| :-------------- | :-------------- |
| 奶及奶制品      | 350-500ml       |
| 饮水量（2-3岁） | 600-700ml       |
| 饮水量（4-5岁） | 700-800ml       |
| 正餐安排        | 3次正餐+2次加餐 |

### 回应式喂养——饥饱信号识别

| 信号类型 | 饥饿表现           | 饱腹表现           |
| :------- | :----------------- | :----------------- |
| 早期     | 觅食反射、吸吮动作 | 吸吮减慢           |
| 中期     | 把手放入嘴里吸吮   | 张嘴、头转开       |
| 晚期     | 哭闹（最晚信号）   | 停止吸吮、推开食物 |
