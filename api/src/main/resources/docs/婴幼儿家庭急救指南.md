## 一、气道异物梗阻（窒息急救）

### 判断是否需要急救

- ✅ **需要急救**：孩子突然无法发声、呼吸困难、面色发紫、用手卡住脖子
- ❌ **不要干预**：孩子还能用力咳嗽、哭出声——鼓励他咳出来

### 1岁以下婴儿：背部拍击+胸部冲击

text

```
【步骤】
1. 大人坐下，前臂托住婴儿，手固定其下颌（手指不堵嘴）
2. 让婴儿脸朝下，头低脚高，趴在你的前臂上
3. 用另一手掌根，在婴儿肩胛骨之间用力拍击5次
4. 如果异物未排出，将婴儿翻转（脸朝上，仍头低脚高）
5. 用两指在婴儿两乳头连线中点下方，快速按压5次（深度约4cm）
6. 重复拍背+压胸，直到异物排出或婴儿失去意识
```



### 1岁以上儿童：海姆立克急救法

text

```
【步骤】
1. 站在孩子身后，双臂环抱其腰部
2. 一手握拳，拳眼对准肚脐上方（约在肚脐与胸骨之间）
3. 另一手包住拳头，快速向内、向上冲击腹部
4. 连续冲击，直到异物排出
```



**口诀：剪刀-石头-布**

- ✌️ 剪刀：肚脐上方两指定位
- ✊ 石头：一手握拳放在定位处
- ✋ 布：另一手包住拳头向内上冲击

### 失去意识时

立即停止冲击，开始心肺复苏（见后文）。

## 二、热性惊厥（发热抽搐）

### 正确做法

| 步骤    | 操作                             |
| :------ | :------------------------------- |
| 1. 侧卧 | 让孩子侧躺，防止口水或呕吐物呛入 |
| 2. 清场 | 移开周围硬物、尖锐物品           |
| 3. 松衣 | 解开衣领，保持呼吸通畅           |
| 4. 计时 | 记录抽搐开始和结束时间           |
| 5. 就医 | 抽搐停止后送医检查               |

### ❌ 绝对禁止

- ❌ 不要往嘴里塞任何东西（手指、毛巾、筷子）
- ❌ 不要掐人中、针刺、拍打
- ❌ 不要强行按压孩子的四肢
- ❌ 抽搐时不要喂水喂药

### 什么情况要打120

- 抽搐超过5分钟
- 短时间内反复抽搐
- 孩子呼吸困难或面色发紫
- 抽搐后意识不清醒

## 三、心肺复苏（CPR）

**⚠️ 仅用于孩子无反应 + 无呼吸（或仅有濒死喘息）**

### 1岁以下婴儿

| 项目      | 操作                         |
| :-------- | :--------------------------- |
| 按压位置  | 两乳头连线中点正下方         |
| 按压方式  | 两指垂直按压                 |
| 按压深度  | 约4厘米（胸廓前后径1/3）     |
| 按压频率  | 100-120次/分                 |
| 按压:吹气 | 30:2（单人）                 |
| 吹气方式  | 口对口鼻（包住婴儿口鼻吹气） |

### 1-8岁儿童

| 项目      | 操作                         |
| :-------- | :--------------------------- |
| 按压位置  | 胸骨下半段（两乳头连线中点） |
| 按压方式  | 单手或双手掌根               |
| 按压深度  | 约5厘米                      |
| 按压频率  | 100-120次/分                 |
| 按压:吹气 | 30:2（单人）                 |
| 吹气方式  | 口对口（捏住鼻子）           |

### 快速记忆

text

```
按压节奏：跟着《最炫民族风》或《小苹果》的节拍（约100-120拍/分）
按压通气比：30次按压 → 2次人工呼吸 → 重复
停止条件：孩子恢复呼吸/心跳，或急救人员到达
```

## 四、烧烫伤

### 五字口诀：冲 → 脱 → 泡 → 盖 → 送

| 步骤   | 操作                                         |
| :----- | :------------------------------------------- |
| **冲** | 流动凉水冲洗伤处 **15-30分钟**（不要用冰）   |
| **脱** | 边冲边轻轻脱去烫伤处的衣物，粘住时剪开勿硬扯 |
| **泡** | 继续在凉水中浸泡15-30分钟                    |
| **盖** | 用干净纱布或保鲜膜覆盖（防感染）             |
| **送** | 严重时送医                                   |

### ❌ 绝对禁止

- ❌ 涂抹牙膏、酱油、醋、香油、面粉
- ❌ 用冰直接敷
- ❌ 挑破水泡

## 五、跌落/头部撞伤

### 观察判断

**立即就医的信号**：

- 意识不清、叫不醒
- 呕吐（尤其是反复呕吐）
- 抽搐
- 从耳朵或鼻孔流出清澈液体或血
- 双眼瞳孔大小不一
- 异常哭闹或异常安静、嗜睡

**在家观察即可**（无以上情况）：

- 冷敷肿包（每次15分钟）
- 观察24小时，注意精神状态变化
- 正常玩耍、进食可放心

## 六、发热处理

### 何时用退烧药

| 条件                     | 建议                   |
| :----------------------- | :--------------------- |
| 体温 ≥38.5℃ + 精神状态差 | 可用退烧药             |
| 体温 ≥39℃                | 可用退烧药             |
| 体温高但精神好、能吃能玩 | 不一定用药，多喝水观察 |

### 退烧药选择

| 药物                   | 适用年龄 | 用法（按体重计算）      | 备注            |
| :--------------------- | :------- | :---------------------- | :-------------- |
| 对乙酰氨基酚（泰诺林） | ≥3个月   | 10-15mg/kg，4-6小时一次 | 24小时不超过4次 |
| 布洛芬（美林）         | ≥6个月   | 5-10mg/kg，6-8小时一次  | 24小时不超过4次 |

### ❌ 禁用

- 阿司匹林（可能引起瑞氏综合征）
- 安乃近、尼美舒利
- 成人退烧药减半给孩子吃

### 物理降温

- 减少衣物（不要捂汗）
- 温水擦身（水温32-34℃，不推荐酒精或冰水）
- 多喝水、多排尿

## 七、鼻出血

### 正确做法

text

```
1. 让孩子坐直，头微微前倾（不要后仰）
2. 拇指和食指捏住鼻翼（鼻子下半部）
3. 持续按压10分钟（中途不要松手看）
4. 用嘴呼吸
```



### ❌ 错误做法

- ❌ 头后仰（血液流进喉咙可能呛咳或呕吐）
- ❌ 举手、举对侧手（无效）
- ❌ 塞纸巾（取出时可能再次出血）

### 需要就医的情况

- 按压20分钟仍不止血
- 反复频繁鼻出血
- 出血量大、面色苍白

## 九、紧急情况快速对照

| 情况               | 第一动作          | 禁忌             | 是否打120                    |
| :----------------- | :---------------- | :--------------- | :--------------------------- |
| 异物卡喉（有意识） | 海姆立克/拍背压胸 | 不要用手抠       | 无效或加重时                 |
| 无反应+无呼吸      | 立即心肺复苏      | 不要犹豫等待     | 复苏同时叫人或先叫（看情况） |
| 热性惊厥           | 侧卧、清场        | 不要塞嘴、掐人中 | 超过5分钟                    |
| 烧烫伤             | 冲凉水15-30分钟   | 不要涂牙膏/酱油  | 严重时                       |
| 头部撞伤+意识不清  | 打120             | 不要随意搬动     | 是                           |
| 发热（≥40℃）       | 退烧药            | 不要捂汗         | 退烧无效且精神差             |
