## 一、常用估算公式

| 项目               | 公式                                   |
| :----------------- | :------------------------------------- |
| **体重**（1-6月）  | 出生体重 + 月龄 × 0.7                  |
| **体重**（7-12月） | 6 + 月龄 × 0.25                        |
| **体重**（2-12岁） | 年龄 × 2 + 8                           |
| **身高**（2-12岁） | 年龄 × 7 + 70                          |
| **头围**           | 出生34cm → 1岁46cm → 2岁48cm → 5岁50cm |

## 五、判断标准（百分位）

| 百分位       | 评价     | 建议       |
| :----------- | :------- | :--------- |
| **< P3**     | 生长迟缓 | 建议就医   |
| **P3 - P97** | 正常范围 | 动态监测   |
| **> P97**    | 需关注   | 排查肥胖等 |

> **核心原则**：单次数值意义有限，重点关注生长曲线趋势。只要曲线在P3-P97之间且平稳上升，即使低于P50也属正常。
