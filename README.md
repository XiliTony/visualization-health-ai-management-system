## 项目简介

本项目是一套完整的**企业级健康管理 + AI 复合系统**，旨在帮助用户全面、便捷地管理个人健康数据。系统支持用户记录心率、血压、睡眠时长等多种健康指标，并允许**自定义健康项**，满足个性化需求。健康记录以**趋势图**和**表格**形式直观展示，支持**导出为 Excel**。每个健康项均设有阈值，超出范围时系统自动**预警**，帮助用户及时发现健康风险。

系统深度集成 **AI 能力**，基于 **Qwen3-Max 大模型**，结合**健康知识库**进行检索增强生成（RAG），并接入**智谱 Web 搜索工具**实现联网搜索，显著提升回答的准确性和时效性。AI 通过 **Function Calling** 技术，支持用户通过**对话方式添加健康项和健康记录**，能够智能判断系统中是否已存在相应健康项，并引导用户完成创建。AI 还能根据已有健康数据生成**健康报告**，提供专业、个性化的健康建议。

## 技术架构

本项目采用 **前后端分离架构**：

### 前端

- **Vue2**：构建用户界面
- **ElementUI**：搭建页面组件
- **ECharts**：展示可视化图表（趋势图等）
- **Vuex**：管理账号信息及全局状态
- **Axios**：实现前后端数据交互

### 后端

- **SpringBoot3**：遵循三层架构（Controller、Service、DAO）
- **LangChain4j**：集成大语言模型开发能力

### AI 能力集成

- **Qwen3-Max**：大语言模型基座
- **RAG（检索增强生成）**：结合健康知识库
- **智谱 Web 搜索**：联网获取最新信息
- **Function Calling**：实现对话式健康数据管理

## 主要功能

| 功能模块 | 说明 |
|---------|------|
| 健康项管理 | 支持预设健康项（心率、血压、睡眠时长等）及用户自定义健康项 |
| 健康记录 | 记录各健康项的数值与时间，支持增删改查 |
| 数据可视化 | 趋势图 + 表格双模式展示，表格支持导出为 Excel |
| 阈值预警 | 每个健康项独立阈值，超出自动警告 |
| AI 智能助手 | 对话式添加健康项/记录，生成健康报告，提供专业建议 |
| 联网搜索增强 | AI 可联网获取最新健康资讯，提升回答准确性 |

## 项目亮点

- **专业化**：AI 结合健康知识库 RAG，建议具备专业可信度
- **智能化**：Function Calling 实现自然语言交互管理健康数据
- **个性化**：支持自定义健康项，满足不同用户需求
- **企业级**：前后端分离，架构清晰，易于扩展与维护
