# 基于可视化实时展示的AI健康管理系统
<img width="674" height="674" alt="logo" src="https://github.com/user-attachments/assets/dccd176b-e042-43af-a990-d299c57cfa3d" />
本项目是一套完整的企业级健康管理+AI复合系统。
可供用户记录自己的各个健康项，例如心率、血压、睡眠时长等等，支持用户自定义健康项，用户的健康记录会用趋势图，表格展示，表格能用Excel导出，每个健康项都有阈值，超出阈值会进行警告，从而使用户很方便地对各个健康数据进行管理。
项目集成了AI功能，使用Qwen3-Max大模型搭配健康知识库进行检索增强生成（RAG），添加了智谱Web搜索工具进行联网搜索，提高了回答的准确性。AI使用FunctionCalling实现了对话添加健康项、健康记录功能，AI能够根据系统内是否存在健康项进行判断，然后对用户进行引导创建内容。AI能够根据系统内的健康数据生成健康报告，给出相应的健康建议，具备良好的专业性。
本项目采用前后端分离架构，前端技术栈使用Vue2构建主体，ElementUI搭建页面，ECharts展示可视化图表，Vuex管理账号信息，Axios进行前后端交互，后端技术栈使用SpringBoot3按照三层架构模式设计，集成LangChain4j框架进行大语言模型开发。
